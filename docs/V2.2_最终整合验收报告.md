## 5. 关键口径复核

- `invalid_test_set`、provider failure、artifact hash mismatch、setup failure、flaky clean replay 不能解释为 0 捕获。
- replay 不调用 LLM，只执行冻结测试集。
- `local_subprocess` 不是安全沙箱，不能宣称 enforce 网络或资源隔离。
- Docker 不可用、setup failure、executor failure 是执行/评测有效性事件，不是策略 0 捕获。
- source context 缺目标源码、缺模型字段、缺 fixture 证据时，必须拒绝、降级或标记 incomplete。
- Contract Guard 是系统边界，不是提示词装饰。
- prompt 只消费 prompt-safe evidence；audit-only 事件默认只进入 report/audit。
- auto mutation v0 是最小 confirmation 链路，不是完整 mutation engine。
